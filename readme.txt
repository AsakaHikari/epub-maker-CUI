☆☆☆epubmakerCUI 使い方☆☆☆

jarファイルを実行します。
そうしたら中にepub化したいファイルをドラッグアンドドロップで入れてください。
そうするとepub化が始まります。
上のラジオボタンで。見開きの方式を選べます。
チェックボックスにチェックがされていると、最初のページを表紙として中央に配置します。(デフォルトでチェックが入っています)


☆☆☆出版社/サークル追加方法☆☆☆
最初に起動すると、「ドキュメント」の中に「epubmakercui」というフォルダができると思います。
その中に「publisherlist.txt」というファイルがあると思います。
これが出版社のリストです。
新しく出版社を追加するには、ファイルの末尾に
(id),(出版社名)
という形式で追加してください。
なお、サークルは出版社と同じものとして扱われます。
