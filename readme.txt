☆☆☆epubmakerCUI 使い方☆☆☆

方法1.そのまま実行
epubにしたいフォルダが入ったフォルダ内でこのepubmakercui.jarを実行すれば、その中の全フォルダをepub化しようと試みます。
この際、id-(何らかの文字列)という名前のフォルダ以外は無視されますが、こういった名前のフォルダで無関係なものは入れないでください(無理やりepub化しようとして誤作動します)。
この方法の場合、すべて右→左の見開きとして生成します。
また、出来上がるepubファイルは
(id).epub
という名前になります。



方法2.コマンドプロンプトから実行

epubにしたいフォルダが入ったフォルダ内にこのepubmakercui.jarを置き、コマンドプロンプトから以下のコマンドで実行してください。
この際、cdコマンドなどでコマンドプロンプトのカレントディレクトリをepubmakercui.jarと同じフォルダにしないと実行できません。

java -jar epubmakercui.jar -type=1(右0、左1、中央2)

-type= の値によって、見開きの設定が変わります。
0なら方法1と同じ右→左の見開き、1なら左→右の見開き、2ならすべて中央です。
